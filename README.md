# mdk-3-course-2024
Учебный проект, выполненный на первом семестре третьего курса.

WPF-приложение, использующее базу данных, созданной с помощью MySQL.

Функционал: просмотр, добавление, редактирование, удаление различных данных из базы данных. Вывод в Excel. Фильтрация, сортировка, вывод информации по страницам. Составление расписаний. Переход на страницу с особым функционалом намеренно интуитивно непонятен, для перехода к ней необходимо нажать на логотип в окне сотрудников.
